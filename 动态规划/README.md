# 动态规划专题面试复习

## 1. 复习方法

参考代码随想录的动规五部曲：

1. 确定 `dp` 数组以及下标含义
2. 确定递推公式
3. 确定初始化
4. 确定遍历顺序
5. 举例推导 `dp` 数组

面试中不要只背递推式，优先说清楚 `dp[i]` 或 `dp[i][j]` 的业务含义。只要含义稳定，初始化、遍历顺序和返回值就能自然推出。

## 3. 分类模板

### 3.1 基础线性 DP

适用题目：`H_70.java`、`H_198.java`

核心特征：当前状态只依赖前面固定几个状态。

```java
int[] dp = new int[n + 1];
dp[0] = 初始值;
dp[1] = 初始值;

for (int i = 2; i <= n; i++) {
    dp[i] = 根据 dp[i - 1], dp[i - 2] 推导;
}
return dp[n];
```

常见含义：

- 爬楼梯：`dp[i]` 是到第 `i` 阶的方法数，递推 `dp[i] = dp[i - 1] + dp[i - 2]`。
- 打家劫舍：`dp[i]` 是考虑到第 `i` 间房的最大金额，递推通常是“偷当前”和“不偷当前”取最大。

面试话术：我把问题定义成前缀最优，当前选择只会影响相邻位置，所以能从前两个状态推出来。

### 3.2 网格路径 DP

适用题目：`H_62.java`、`H_64.java`、`H_221.java`

核心特征：二维坐标状态，从上方、左方或左上方转移。

```java
int[][] dp = new int[m][n];
// 初始化第一行、第一列

for (int i = 1; i < m; i++) {
    for (int j = 1; j < n; j++) {
        dp[i][j] = 根据 dp[i - 1][j], dp[i][j - 1], dp[i - 1][j - 1] 推导;
    }
}
```

常见含义：

- 不同路径：`dp[i][j]` 是走到 `(i,j)` 的路径数。
- 最小路径和：`dp[i][j]` 是走到 `(i,j)` 的最小路径和。
- 最大正方形：`dp[i][j]` 是以原矩阵 `matrix[i - 1][j - 1]` 为右下角的最大正方形边长；代码里通常会多开一行一列做哨兵。

面试易错点：第一行、第一列通常要单独初始化；最大正方形需要看上、左、左上三个方向的最小值。

### 3.3 连续子数组 DP

适用题目：`H_53.java`、`H_152.java`

核心特征：`dp[i]` 一般表示“必须以 `i` 结尾”的答案，最终答案不一定是 `dp[n - 1]`，而是遍历过程中的最大值。

```java
int[] dp = new int[nums.length];
dp[0] = nums[0];
int result = nums[0];

for (int i = 1; i < nums.length; i++) {
    dp[i] = Math.max(dp[i - 1] + nums[i], nums[i]);
    result = Math.max(result, dp[i]);
}
return result;
```

常见含义：

- 最大子数组和：`dp[i]` 是以 `nums[i]` 结尾的最大连续和。
- 乘积最大子数组：`dp[i][0]` 是以 `nums[i]` 结尾的最大乘积，`dp[i][1]` 是以 `nums[i]` 结尾的最小乘积；负数会翻转最大/最小，所以两个状态都要维护。

面试话术：连续子数组要求不能断，所以状态定义成“以当前位置结尾”，每一步只有接上前一段或从当前重新开始两种选择。

### 3.4 01 背包

适用题目：`H_416.java`、`H_494.java`

核心特征：每个物品只能使用一次。一维滚动数组必须先遍历物品，再倒序遍历容量，避免同一个物品被重复使用。

```java
int[] dp = new int[target + 1];

for (int i = 0; i < nums.length; i++) {
    for (int j = target; j >= nums[i]; j--) {
        dp[j] = 根据 dp[j] 和 dp[j - nums[i]] 推导;
    }
}
```

两种常见问法：

- 最大价值：`dp[j] = Math.max(dp[j], dp[j - weight] + value)`。
- 方案数：`dp[j] += dp[j - nums[i]]`，并且 `dp[0] = 1`。

题目转换：

- 分割等和子集：目标是凑出总和的一半，`dp[j]` 表示容量 `j` 下最大能凑出的和。
- 目标和：设加号集合和为 `x`，总和为 `sum`，有 `x - (sum - x) = target`，得到 `x = (target + sum) / 2`，转成装满背包的方案数。

面试易错点：

- `target + sum` 为奇数时无解。
- `Math.abs(target) > sum` 时无解。
- 01 背包一维数组容量必须倒序。

### 3.5 完全背包

适用题目：`H_322.java`、`H_279.java`，`H_139.java` 可用完全背包思想理解。

核心特征：每个物品可以使用无限次。一维数组容量通常正序遍历，让当前物品能被重复使用。

```java
int[] dp = new int[amount + 1];
Arrays.fill(dp, INF);
dp[0] = 0;

for (int i = 0; i < coins.length; i++) {
    for (int j = coins[i]; j <= amount; j++) {
        if (dp[j - coins[i]] != INF) {
            dp[j] = Math.min(dp[j], dp[j - coins[i]] + 1);
        }
    }
}
```

常见含义：

- 零钱兑换：`dp[j]` 是组成金额 `j` 的最少硬币数。
- 完全平方数：把每个平方数当成物品，`dp[j]` 是组成 `j` 的最少物品数。
- 单词拆分：`dp[i]` 是前 `i` 个字符是否能拆分，枚举切分点 `j`。

遍历顺序口诀：

- 01 背包一维：物品外层，容量倒序。
- 完全背包求组合数：物品外层，容量正序。
- 完全背包求排列数：容量外层，物品内层。
- 求最小个数：多数情况下物品、容量顺序都能推导，但容量必须正序以允许重复使用。

### 3.6 股票状态机

适用题目：`H_121.java`、`H_309.java`

核心特征：每天的利润由“持股状态”和“不持股状态”转移而来。状态定义比递推公式更重要。

一次买卖模板：

```java
int[][] dp = new int[prices.length][2];
dp[0][0] = -prices[0]; // 持股
dp[0][1] = 0;          // 不持股

for (int i = 1; i < prices.length; i++) {
    dp[i][0] = Math.max(dp[i - 1][0], -prices[i]);
    dp[i][1] = Math.max(dp[i - 1][1], dp[i - 1][0] + prices[i]);
}
return dp[prices.length - 1][1];
```

冷冻期状态：

- `dp[i][0]`：第 `i` 天持股。
- `dp[i][1]`：第 `i` 天保持不持股。
- `dp[i][2]`：第 `i` 天卖出。
- `dp[i][3]`：第 `i` 天处于冷冻期。

面试话术：股票题我先列状态，再列每个状态由哪些前一天状态转移。只要状态互斥且覆盖所有情况，递推式就不会漏。

### 3.7 子序列和双序列 DP

适用题目：`H_300.java`、`H_718.java`、`H_1143.java`、`H_72.java`、`H_10.java`

核心特征：

- 单序列：通常枚举 `i`，再枚举 `j < i`。
- 双序列：通常定义 `dp[i][j]` 表示两个前缀之间的关系。
- 连续问题看左上角 `dp[i - 1][j - 1]`；非连续问题还要看 `dp[i - 1][j]` 和 `dp[i][j - 1]`。

LIS 模板：

```java
Arrays.fill(dp, 1);
int result = 1;
for (int i = 1; i < nums.length; i++) {
    for (int j = 0; j < i; j++) {
        if (nums[i] > nums[j]) {
            dp[i] = Math.max(dp[i], dp[j] + 1);
        }
    }
    result = Math.max(result, dp[i]);
}
```

最长公共子序列模板：

```java
int[][] dp = new int[m + 1][n + 1];
for (int i = 1; i <= m; i++) {
    for (int j = 1; j <= n; j++) {
        if (a[i - 1] == b[j - 1]) {
            dp[i][j] = dp[i - 1][j - 1] + 1;
        } else {
            dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
        }
    }
}
```

编辑距离模板：

```java
int[][] dp = new int[m + 1][n + 1];
for (int i = 0; i <= m; i++) dp[i][0] = i;
for (int j = 0; j <= n; j++) dp[0][j] = j;

for (int i = 1; i <= m; i++) {
    for (int j = 1; j <= n; j++) {
        if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
            dp[i][j] = dp[i - 1][j - 1];
        } else {
            dp[i][j] = Math.min(
                Math.min(dp[i - 1][j], dp[i][j - 1]),
                dp[i - 1][j - 1]
            ) + 1;
        }
    }
}
```

面试易错点：

- `dp[i][j]` 使用前缀含义时，字符串访问是 `i - 1` 和 `j - 1`。
- 最长重复子数组是连续的，不相等时自然为 `0`。
- 最长公共子序列不要求连续，不相等时取上方和左方最大值。

### 3.8 回文区间 DP

适用题目：`H_5.java`、`双指针/H_647.java`

核心特征：判断一个真实下标区间 `s[i..j]` 是否是回文串。这里的 `i`、`j` 是字符串真实下标，不是“前 `i` 个字符”的前缀含义。

```java
boolean[][] dp = new boolean[n][n];

for (int i = n - 1; i >= 0; i--) {
    for (int j = i; j < n; j++) {
        if (s.charAt(i) == s.charAt(j)) {
            if (j - i <= 1) {
                dp[i][j] = true;
            } else {
                dp[i][j] = dp[i + 1][j - 1];
            }
        }
    }
}
```

状态含义：

- `dp[i][j] == true`：`s[i..j]` 是回文串。
- `dp[i][j] == false`：`s[i..j]` 不是回文串。

递推逻辑：

- 如果 `s[i] != s[j]`，两端不同，`s[i..j]` 一定不是回文。
- 如果 `s[i] == s[j]` 且 `j - i <= 1`，长度为 1 或 2，直接是回文。
- 如果 `s[i] == s[j]` 且 `j - i > 1`，继续看中间 `dp[i + 1][j - 1]`。

遍历顺序：

- 因为 `dp[i][j]` 依赖 `dp[i + 1][j - 1]`，所以 `i` 必须从后往前遍历。
- `j` 是右边界，从 `i` 往后遍历到 `n - 1`，循环条件是 `j < n`。

两个题的区别：

- 最长回文子串：每次 `dp[i][j]` 为 `true` 时，用 `j - i + 1` 更新最长长度和起点。
- 回文子串：每次 `dp[i][j]` 为 `true` 时，计数器加一。

面试易错点：

- 外层 `i` 从后往前时，循环变量要 `i--`，不是 `i++`。
- 内层 `j` 要遍历到最后一个字符，条件是 `j < n`，不是 `j < n - 1`。
- Java `substring(start, end)` 的第二个参数是结束下标，不是长度；最长回文子串返回 `s.substring(start, start + maxLen)`。

### 3.9 区间 DP

适用题目：`H_312.java`

核心特征：按区间长度从小到大枚举，最后枚举区间内的分割点。

```java
for (int len = 3; len <= n; len++) {
    for (int left = 0; left + len - 1 < n; left++) {
        int right = left + len - 1;
        for (int k = left + 1; k < right; k++) {
            dp[left][right] = Math.max(
                dp[left][right],
                dp[left][k] + dp[k][right] + nums[left] * nums[k] * nums[right]
            );
        }
    }
}
```

戳气球关键含义：`dp[i][j]` 表示开区间 `(i,j)` 里的气球全部戳完能得到的最大金币。枚举 `k` 时，不是第一个戳 `k`，而是最后一个戳 `k`，这样左右区间才互不影响。

## 4. 每题复述

### H_70 - 爬楼梯

- `dp[i]`：到第 `i` 阶的方法数。
- 初始化：`dp[1] = 1`，`dp[2] = 2`。
- 递推：最后一步从 `i - 1` 跨 1 阶，或从 `i - 2` 跨 2 阶，所以 `dp[i] = dp[i - 1] + dp[i - 2]`。
- 面试话术：这是斐波那契模型，状态表示到达当前台阶的方法数。

### H_62 - 不同路径

- `dp[i][j]`：到达 `(i,j)` 的路径数。
- 初始化：第一行和第一列都只能一直向右或向下走，所以为 `1`。
- 递推：`dp[i][j] = dp[i - 1][j] + dp[i][j - 1]`。
- 面试话术：机器人只能从上方或左方到达当前位置，所以路径数相加。

### H_5 - 最长回文子串

- `dp[i][j]`：真实下标区间 `s[i..j]` 是否是回文串。
- 初始化：默认都是 `false`；长度为 1 的子串一定是回文，可以通过 `j - i <= 1` 在遍历时处理。
- 递推：如果 `s[i] == s[j]`，并且 `j - i <= 1` 或 `dp[i + 1][j - 1] == true`，则 `dp[i][j] = true`。
- 遍历：`i` 从后往前，`j` 从 `i` 往后，因为 `dp[i][j]` 依赖 `dp[i + 1][j - 1]`。
- 更新答案：发现 `dp[i][j] == true` 时，当前长度是 `j - i + 1`；如果更长，就更新 `start = i`、`maxLen = j - i + 1`。
- 返回：`s.substring(start, start + maxLen)`，注意第二个参数是结束下标，不是长度。
- 面试话术：回文串两端字符必须相同，中间也必须是回文；我用二维 DP 记录每个区间是否回文，再维护最长区间。

### H_53 - 最大子数组和

- `dp[i]`：以 `nums[i]` 结尾的最大连续子数组和。
- 初始化：`dp[0] = nums[0]`。
- 递推：`dp[i] = Math.max(dp[i - 1] + nums[i], nums[i])`。
- 返回：所有 `dp[i]` 的最大值。
- 面试话术：因为要求连续，以当前位置结尾时只有接前面或从当前重新开始。

### H_152 - 乘积最大子数组

- `dp[i][0]`：以 `nums[i]` 结尾的连续子数组的最大乘积。
- `dp[i][1]`：以 `nums[i]` 结尾的连续子数组的最小乘积。
- 初始化：`dp[0][0] = nums[0]`，`dp[0][1] = nums[0]`。
- 递推：当前位置有三种选择，分别是只选 `nums[i]`、接上前一个最大乘积、接上前一个最小乘积。
- 公式：`dp[i][0] = max(nums[i], dp[i - 1][0] * nums[i], dp[i - 1][1] * nums[i])`；`dp[i][1] = min(nums[i], dp[i - 1][0] * nums[i], dp[i - 1][1] * nums[i])`。
- 返回：遍历过程中所有 `dp[i][0]` 的最大值，不能直接返回 `dp[n - 1][0]`。
- 面试话术：乘积遇到负数会让最大值和最小值互换，所以除了最大乘积，还必须保留最小乘积；`dp[i][0]` 只代表以当前位置结尾，最终答案要用 `result` 全局维护。

### H_416 - 分割等和子集

- 转换：数组总和为奇数直接返回 `false`；否则目标是凑出 `sum / 2`。
- `dp[j]`：容量为 `j` 时能凑出的最大子集和。
- 递推：`dp[j] = Math.max(dp[j], dp[j - nums[i]] + nums[i])`。
- 遍历：外层物品，内层容量倒序。
- 返回：`dp[target] == target`。
- 面试话术：每个数只能用一次，所以是 01 背包；如果半和能被凑满，就能分成两个等和子集。

### H_494 - 目标和

- 转换：正数集合和为 `x`，负数集合和为 `sum - x`，所以 `x = (target + sum) / 2`。
- `dp[j]`：凑满和为 `j` 的方法数。
- 初始化：`dp[0] = 1`，表示什么都不选凑出 0 有一种方法。
- 递推：`dp[j] += dp[j - nums[i]]`。
- 遍历：01 背包倒序。
- 面试话术：加减号问题可以转成从数组中选一些数凑指定和，问有多少种选法。

### H_322 - 零钱兑换

- `dp[j]`：凑成金额 `j` 的最少硬币数。
- 初始化：`dp[0] = 0`，其他位置为无穷大。
- 递推：`dp[j] = Math.min(dp[j], dp[j - coin] + 1)`。
- 遍历：硬币外层，金额正序。
- 面试话术：硬币可以重复使用，所以是完全背包；求最少硬币数，状态取最小值。

### H_279 - 完全平方数

- `dp[j]`：组成整数 `j` 的最少完全平方数个数。
- 物品：`1, 4, 9, ...`。
- 递推：`dp[j] = Math.min(dp[j], dp[j - i * i] + 1)`。
- 遍历：平方数外层，容量正序。
- 面试话术：把每个平方数看作可以无限使用的物品，就是完全背包求最少物品数。

### H_139 - 单词拆分

- `dp[i]`：字符串前 `i` 个字符是否可以被字典拆分。
- 初始化：`dp[0] = true`，空串可以被拆分。
- 递推：存在 `j < i`，如果 `dp[j]` 为真且 `s[j..i)` 在字典中，则 `dp[i] = true`。
- 面试话术：枚举最后一个单词的起点，只要前缀可拆且最后一段在字典里，当前前缀就可拆。
- 优化点：把 `wordDict` 转成 `HashSet`，避免 `List.contains` 每次线性查找。

### H_121 - 买卖股票的最佳时机

- `dp[i][0]`：第 `i` 天持股时的最大利润。
- `dp[i][1]`：第 `i` 天不持股时的最大利润。
- 初始化：`dp[0][0] = -prices[0]`，`dp[0][1] = 0`。
- 递推：`dp[i][0] = Math.max(dp[i - 1][0], -prices[i])`；`dp[i][1] = Math.max(dp[i - 1][1], dp[i - 1][0] + prices[i])`。
- 面试话术：只能买卖一次，所以持股状态只能来自之前持股或今天第一次买入。

### H_309 - 最佳买卖股票时机含冷冻期

- `dp[i][0]`：持股。
- `dp[i][1]`：保持不持股。
- `dp[i][2]`：今天卖出。
- `dp[i][3]`：今天冷冻期。
- 关键递推：卖出只能从昨天持股来，冷冻期只能从昨天卖出来，买入不能从昨天卖出直接来。
- 面试话术：冷冻期题用多个互斥状态描述每天结束后的状态，避免用模糊的“不持股”漏掉限制。

### H_300 - 最长递增子序列

- `dp[i]`：以 `nums[i]` 结尾的 LIS 长度。
- 初始化：每个元素自己都是长度为 `1` 的递增子序列。
- 递推：如果 `nums[i] > nums[j]`，则 `dp[i] = Math.max(dp[i], dp[j] + 1)`。
- 返回：所有 `dp[i]` 的最大值。
- 面试话术：我枚举当前位置作为结尾，再找它前面能接上的元素。

### H_718 - 最长重复子数组

- `dp[i][j]`：以 `nums1[i - 1]` 和 `nums2[j - 1]` 结尾的公共连续后缀长度。
- 初始化：多开一行一列为 `0`。
- 递推：相等时 `dp[i][j] = dp[i - 1][j - 1] + 1`，不相等时保持 `0`。
- 返回：遍历过程中的最大值。
- 面试话术：因为要求连续，所以只看左上角；一旦不相等，当前结尾的连续长度断掉。

### H_1143 - 最长公共子序列

- 代码随想录定义：`dp[i][j]` 表示长度为 `[0, i - 1]` 的字符串 `text1` 与长度为 `[0, j - 1]` 的字符串 `text2` 的最长公共子序列。
- 等价面试理解：`dp[i][j]` 表示 `text1` 的前 `i` 个字符 `text1[0..i)` 和 `text2` 的前 `j` 个字符 `text2[0..j)` 的最长公共子序列长度。
- 下标注意：`i`、`j` 表示前缀长度，访问原字符串时用 `i - 1` 和 `j - 1`。
- 递推：字符相等取 `dp[i - 1][j - 1] + 1`，不相等取 `Math.max(dp[i - 1][j], dp[i][j - 1])`。
- 面试话术：子序列不要求连续，所以不相等时可以分别丢掉一个字符，看哪个前缀答案更大。

### H_72 - 编辑距离

- 代码随想录定义：`dp[i][j]` 表示以下标 `i - 1` 为结尾的字符串 `word1`，和以下标 `j - 1` 为结尾的字符串 `word2`，最近编辑距离为 `dp[i][j]`。
- 等价面试理解：`dp[i][j]` 表示 `word1` 的前 `i` 个字符 `word1[0..i)` 转成 `word2` 的前 `j` 个字符 `word2[0..j)` 的最少操作数。
- 下标注意：`i`、`j` 表示前缀长度，访问原字符串时用 `i - 1` 和 `j - 1`。
- 初始化：空串转成长度 `j` 需要插入 `j` 次；长度 `i` 转空串需要删除 `i` 次。
- 递推：字符相等不用操作；不相等时从删除、插入、替换三种操作里取最小值加一。
- 面试话术：编辑距离的三个来源分别对应删除 `word1[i-1]`、插入 `word2[j-1]`、替换当前字符。

### H_647 - 回文子串

- 当前文件位置：`双指针/H_647.java`，也可以用 DP 复习。
- `dp[i][j]`：真实下标区间 `s[i..j]` 是否是回文串。
- 递推：和 `H_5` 完全一样，两端相等且中间是回文时，当前区间是回文。
- 和 `H_5` 的区别：`H_5` 是求最长回文子串，更新最长长度；`H_647` 是统计所有回文子串，遇到 `dp[i][j] == true` 就计数加一。
- 面试话术：这题不需要记录最长起点，只需要统计有多少个区间满足回文条件。

## 5. 面试高频易错点

- `dp` 含义必须包含“是否以当前位置结尾”“是否是前缀”“是否必须装满”等限定词。
- 回文区间 DP 的 `dp[i][j]` 是真实下标区间，不是前缀长度；遍历时 `i` 从后往前，`j` 从 `i` 往后。
- 一维 01 背包容量倒序；完全背包容量正序。
- 求方案数时通常 `dp[0] = 1`，表示空方案。
- 求最小值时初始化为无穷大，转移前要判断前置状态是否可达。
- 子序列题要区分连续和不连续：连续只看左上，不连续还看上和左。
- 股票题不要只背公式，先列每天结束后的状态。
- 二维字符串 DP 多开一行一列，可以大幅减少边界判断。
- 最终答案不一定是最后一个状态：最大子数组和、乘积最大子数组、LIS、最长重复子数组都要维护全局最大值。
