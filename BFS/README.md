## 4. 易错点

- 网格 BFS/DFS 要先检查边界，再访问数组。
- 遍历过的位置要及时标记，避免重复入队或递归死循环。
- 多源 BFS 要先把所有起点入队，再开始按层扩散。
- 拓扑排序最后要判断是否处理了全部节点。
