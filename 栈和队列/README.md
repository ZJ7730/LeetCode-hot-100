## 3. 推荐刷题顺序

| 顺序 | 题目 | 目标 |
| ---: | --- | --- |
| 1 | `20. 有效的括号` | 掌握栈匹配 |
| 2 | `155. 最小栈` | 掌握辅助栈同步维护最小值 |
| 3 | `394. 字符串解码` | 掌握数字、字符串、括号嵌套解析 |
| 4 | `239. 滑动窗口最大值` | 掌握单调队列 |
| 5 | `295. 数据流的中位数` | 掌握大顶堆 + 小顶堆平衡 |

## 4. 易错点

- 栈题要先确定栈中保存字符、下标、状态还是中间结果。
- 单调队列要在入队前移除破坏单调性的元素。
- 双堆要保持大小差不超过 1，并保证左半部分都小于右半部分。
