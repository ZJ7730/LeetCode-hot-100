## 5. 每题笔记

### H_136 - 只出现一次的数字

- 题目定位：数组中只有一个数字出现一次，其余数字都出现两次，找出这个只出现一次的数字。
- 当前代码思路：把所有元素依次异或到一个变量里，成对出现的数字会被抵消，最后剩下的就是答案。
- 核心性质：`a ^ a = 0`，`a ^ 0 = a`，异或满足交换律和结合律。
- 复杂度：时间 `O(n)`，空间 `O(1)`。

### H_338 - 比特位计数

- 题目定位：给定 `n`，返回 `0` 到 `n` 每个数的二进制中 1 的个数。
- 当前代码思路：`dp[i] = dp[i >> 1] + (i & 1)`，即去掉最低位后，剩余部分的 1 的个数再加上最低位是否为 1。
- 状态定义：`dp[i]` 表示数字 `i` 的二进制中 1 的个数。
- 复杂度：时间 `O(n)`，空间 `O(n)`。
