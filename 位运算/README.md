## 2. 面试高频题

| 优先级 | 题目 | 高频原因 |
| ---: | --- | --- |
| 1 | `136. 只出现一次的数字` | 异或性质基础题 |
| 2 | `338. 比特位计数` | 位运算结合 DP，常用于考察状态转移 |
| 3 | `461. 汉明距离` | 异或后统计 1 的个数，简单但常见 |

## 3. 推荐刷题顺序

| 顺序 | 题目 | 目标 |
| ---: | --- | --- |
| 1 | `136. 只出现一次的数字` | 掌握 `a ^ a = 0` 和 `a ^ 0 = a` |
| 2 | `461. 汉明距离` | 掌握异或找不同位 |
| 3 | `338. 比特位计数` | 掌握 `dp[i] = dp[i >> 1] + (i & 1)` |

## 4. 易错点

- 异或满足交换律和结合律，适合成对抵消。
- 统计二进制 1 可以用 `x &= x - 1`。
- Java 中注意有符号整数右移和无符号右移的区别。
