## 5. 每题笔记

### H_136 - 只出现一次的数字

- 题目定位：数组中只有一个数字出现一次，其余数字都出现两次，找出这个只出现一次的数字。
- 当前代码思路：把所有元素依次异或到一个变量里，成对出现的数字会被抵消，最后剩下的就是答案。
- 核心性质：`a ^ a = 0`，`a ^ 0 = a`，异或满足交换律和结合律。
- 复杂度：时间 `O(n)`，空间 `O(1)`。

### H_338 - 比特位计数

- 题目定位：给定 `n`，返回 `0` 到 `n` 每个数的二进制中 1 的个数。
- 当前代码思路：`dp[i] = dp[i >> 1] + (i & 1)`，即去掉最低位后，剩余部分的 1 的个数再加上最低位是否为 1。
- 状态定义：`dp[i]` 表示数字 `i` 的二进制中 1 的个数。
- 复杂度：时间 `O(n)`，空间 `O(n)`。

### H_461 - 汉明距离

- 题目定位：计算两个整数二进制表示中，对应位置不同的个数。
- 当前代码思路：先计算 `x ^ y`，异或结果中为 `1` 的位置，就是 `x` 和 `y` 不同的位置。
- 关键变量：`res` 保存异或结果；`count` 记录已经统计到的不同位数量。
- 解题步骤：当 `res != 0` 时，不断执行 `res = res & (res - 1)`，每次都会消掉二进制中最右边的一个 `1`，同时 `count++`。
- 核心技巧：`n & (n - 1)` 可以把 `n` 最低位的 `1` 变成 `0`，所以循环次数正好等于 `1` 的个数。
- 复杂度：时间 `O(k)`，`k` 是异或结果中 `1` 的个数；空间 `O(1)`。
- 面试易错点：汉明距离不是比较十进制大小，而是比较二进制每一位是否不同。
- 可复述话术：先异或把不同位标成 `1`，再用 `res & (res - 1)` 一个一个消掉这些 `1`，消掉几次就有几个不同位。
