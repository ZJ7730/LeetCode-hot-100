## 4. 易错点

- 贪心题要能解释局部选择为什么不会影响全局最优。
- 区间题通常先排序，排序字段决定解法是否成立。
- 跳跃游戏 II 中遇到当前边界才增加步数，不是每一步都增加。

## 5. 每题笔记

### H_45 - 跳跃游戏 II

- 题目定位：从数组起点跳到终点，求最少跳跃次数。
- 当前代码思路：把每一次跳跃看成一层，`end` 表示当前这一步能覆盖到的最远边界，`farthest` 表示下一步能到的最远位置。
- 关键变量：`steps` 记录跳跃次数；`end` 是当前层边界；`farthest` 是下一层边界。
- 解题步骤：遍历数组时不断更新 `farthest`；一旦走到 `end`，说明当前层遍历完了，必须再跳一次，并把 `end` 更新为 `farthest`。
- 复杂度：时间 `O(n)`，空间 `O(1)`。
- 面试易错点：跳跃次数不是每移动一步就加，而是每扩展完一层才加。
- 可复述话术：我把每次跳跃看成 BFS 的一层，在当前层里找到下一层最远能到的位置，走到边界时再统一加一步。

### H_763 - 划分字母区间

- 题目定位：把字符串分成尽可能多的片段，要求每个字母只出现在一个片段中。
- 当前代码思路：先记录每个字符最后出现的位置，再扫描字符串维护当前片段的最远边界。
- 关键变量：`last` 记录字符最后出现位置；`start` 是当前片段起点；`end` 是当前片段应到达的最远边界。
- 解题步骤：先遍历一遍字符串得到每个字符最后位置；第二遍扫描时不断更新 `end`；当 `i == end` 时切分一个片段。
- 复杂度：时间 `O(n)`，空间 `O(1)`。
- 面试易错点：切分点一定是当前片段内所有字符的最后出现位置都已经被覆盖的位置。
- 可复述话术：先知道每个字符最后会出现在哪里，再从左到右扫描，当前片段只要还没覆盖完就不能切。
