# 回溯解题模板

## 1. 回溯通用框架

回溯本质是在递归树上做三件事：

1. 做选择
2. 进入下一层
3. 撤销选择

通用代码结构：

```java
List<List<Integer>> result = new ArrayList<>();
List<Integer> path = new ArrayList<>();

void backtrack(参数) {
    if (终止条件) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (选择 : 当前层可选范围) {
        path.add(选择);
        backtrack(下一层参数);
        path.remove(path.size() - 1);
    }
}
```

核心问题：

- 当前层能选什么？
- 什么时候收集答案？
- 下一层从哪里开始？
- 回溯时要恢复哪些状态？

## 2. 子集模板

适用题目：`78. 子集`

特点：每一个递归节点都是一个合法答案，所以进入递归后先收集。

```java
List<List<Integer>> result = new ArrayList<>();
List<Integer> path = new ArrayList<>();

public List<List<Integer>> subsets(int[] nums) {
    backtrack(nums, 0);
    return result;
}

private void backtrack(int[] nums, int startIndex) {
    result.add(new ArrayList<>(path));

    for (int i = startIndex; i < nums.length; i++) {
        path.add(nums[i]);
        backtrack(nums, i + 1);
        path.remove(path.size() - 1);
    }
}
```

记忆点：

- 子集没有固定长度，当前 `path` 随时都可以收集。
- 用 `startIndex` 保证只向后选，避免 `[1,2]` 和 `[2,1]` 重复。
- 时间复杂度 `O(n * 2^n)`。

## 3. 子集去重模板

适用题目：`90. 子集 II`

输入有重复元素时，先排序，再跳过同层重复。

```java
Arrays.sort(nums);

private void backtrack(int[] nums, int startIndex) {
    result.add(new ArrayList<>(path));

    for (int i = startIndex; i < nums.length; i++) {
        if (i > startIndex && nums[i] == nums[i - 1]) {
            continue;
        }

        path.add(nums[i]);
        backtrack(nums, i + 1);
        path.remove(path.size() - 1);
    }
}
```

关键判断：

```java
if (i > startIndex && nums[i] == nums[i - 1]) {
    continue;
}
```

含义：同一层里，相同值只让第一个分支继续搜索。

## 4. 组合模板

适用题目：`77. 组合`

特点：只在路径长度满足要求时收集答案。

```java
List<List<Integer>> result = new ArrayList<>();
List<Integer> path = new ArrayList<>();

public List<List<Integer>> combine(int n, int k) {
    backtrack(n, k, 1);
    return result;
}

private void backtrack(int n, int k, int startIndex) {
    if (path.size() == k) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = startIndex; i <= n; i++) {
        path.add(i);
        backtrack(n, k, i + 1);
        path.remove(path.size() - 1);
    }
}
```

记忆点：

- 组合不关心顺序，所以必须用 `startIndex`。
- 普通不可重复选择，递归传 `i + 1`。
- 如果还需要剪枝，循环上界可以改成 `n - (k - path.size()) + 1`。

剪枝版：

```java
for (int i = startIndex; i <= n - (k - path.size()) + 1; i++) {
    path.add(i);
    backtrack(n, k, i + 1);
    path.remove(path.size() - 1);
}
```

## 5. 组合总和模板

### 5.1 元素可以重复使用

适用题目：`39. 组合总和`

关键：递归传 `i`，表示当前元素下一层还能继续选。

```java
private void backtrack(int[] candidates, int target, int sum, int startIndex) {
    if (sum == target) {
        result.add(new ArrayList<>(path));
        return;
    }
    if (sum > target) {
        return;
    }

    for (int i = startIndex; i < candidates.length; i++) {
        path.add(candidates[i]);
        backtrack(candidates, target, sum + candidates[i], i);
        path.remove(path.size() - 1);
    }
}
```

### 5.2 每个下标只能使用一次

适用题目：`40. 组合总和 II`

关键：递归传 `i + 1`，并且排序后做同层去重。

```java
Arrays.sort(candidates);

private void backtrack(int[] candidates, int target, int sum, int startIndex) {
    if (sum == target) {
        result.add(new ArrayList<>(path));
        return;
    }
    if (sum > target) {
        return;
    }

    for (int i = startIndex; i < candidates.length; i++) {
        if (i > startIndex && candidates[i] == candidates[i - 1]) {
            continue;
        }

        path.add(candidates[i]);
        backtrack(candidates, target, sum + candidates[i], i + 1);
        path.remove(path.size() - 1);
    }
}
```

## 6. 排列模板

适用题目：`46. 全排列`

特点：排列关心顺序，不能用 `startIndex`，每一层都从头枚举，用 `used[]` 排除已经在路径里的元素。

```java
List<List<Integer>> result = new ArrayList<>();
List<Integer> path = new ArrayList<>();

public List<List<Integer>> permute(int[] nums) {
    backtrack(nums, new boolean[nums.length]);
    return result;
}

private void backtrack(int[] nums, boolean[] used) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) {
            continue;
        }

        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, used);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

记忆点：

- 组合和子集用 `startIndex`。
- 排列不用 `startIndex`，用 `used[]`。
- 时间复杂度 `O(n * n!)`。

## 7. 排列去重模板

适用题目：`47. 全排列 II`

先排序，再跳过同层重复分支。

```java
Arrays.sort(nums);

private void backtrack(int[] nums, boolean[] used) {
    if (path.size() == nums.length) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int i = 0; i < nums.length; i++) {
        if (used[i]) {
            continue;
        }
        if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) {
            continue;
        }

        used[i] = true;
        path.add(nums[i]);
        backtrack(nums, used);
        path.remove(path.size() - 1);
        used[i] = false;
    }
}
```

关键判断：

```java
if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) {
    continue;
}
```

含义：前一个相同值没有在当前路径中使用，说明当前是在同一层开启重复分支，要跳过。

## 8. 括号生成模板

适用题目：`22. 括号生成`

特点：不是从数组里选，而是每一层决定放左括号还是右括号。

```java
List<String> result = new ArrayList<>();
StringBuilder path = new StringBuilder();

public List<String> generateParenthesis(int n) {
    backtrack(0, 0, n);
    return result;
}

private void backtrack(int left, int right, int n) {
    if (path.length() == 2 * n) {
        result.add(path.toString());
        return;
    }

    if (left < n) {
        path.append('(');
        backtrack(left + 1, right, n);
        path.deleteCharAt(path.length() - 1);
    }

    if (right < left) {
        path.append(')');
        backtrack(left, right + 1, n);
        path.deleteCharAt(path.length() - 1);
    }
}
```

记忆点：

- `left < n`：左括号还没用完，可以放。
- `right < left`：右括号数量必须小于左括号数量，才能放右括号。
- 合法括号的任意前缀都必须满足 `right <= left`。

## 9. 字符串按位构造模板

适用题目：`17. 电话号码的字母组合`

特点：递归参数 `index` 表示当前处理到字符串的哪一位。

```java
List<String> result = new ArrayList<>();
StringBuilder path = new StringBuilder();

private void backtrack(String digits, int index) {
    if (index == digits.length()) {
        result.add(path.toString());
        return;
    }

    String letters = getLetters(digits.charAt(index));
    for (int i = 0; i < letters.length(); i++) {
        path.append(letters.charAt(i));
        backtrack(digits, index + 1);
        path.deleteCharAt(path.length() - 1);
    }
}
```

记忆点：

- 每一层只负责当前位能选什么。
- 下一层固定处理 `index + 1`。
- 空输入通常直接返回空集合。

## 10. 分割模板

适用题目：`131. 分割回文串`

特点：`startIndex` 表示下一段字符串从哪里开始切。

```java
List<List<String>> result = new ArrayList<>();
List<String> path = new ArrayList<>();

private void backtrack(String s, int startIndex) {
    if (startIndex == s.length()) {
        result.add(new ArrayList<>(path));
        return;
    }

    for (int end = startIndex; end < s.length(); end++) {
        if (!isPalindrome(s, startIndex, end)) {
            continue;
        }

        path.add(s.substring(startIndex, end + 1));
        backtrack(s, end + 1);
        path.remove(path.size() - 1);
    }
}
```

记忆点：

- 切割问题枚举的是结束位置 `end`。
- 当前段合法，才继续递归后面的字符串。
- 递归下一层从 `end + 1` 开始。

## 11. 棋盘和网格 DFS 模板

适用题目：`79. 单词搜索`、`51. N 皇后`

### 11.1 网格搜索

```java
private boolean dfs(char[][] board, String word, int x, int y, int index) {
    if (x < 0 || x >= board.length || y < 0 || y >= board[0].length) {
        return false;
    }
    if (board[x][y] != word.charAt(index)) {
        return false;
    }
    if (index == word.length() - 1) {
        return true;
    }

    char old = board[x][y];
    board[x][y] = '#';

    boolean found =
            dfs(board, word, x + 1, y, index + 1)
            || dfs(board, word, x - 1, y, index + 1)
            || dfs(board, word, x, y + 1, index + 1)
            || dfs(board, word, x, y - 1, index + 1);

    board[x][y] = old;
    return found;
}
```

记忆点：

- 先判断越界，再访问数组。
- 用原地标记或 `visited` 防止当前路径重复使用同一个格子。
- 返回前必须恢复现场。

### 11.2 棋盘放置

```java
private void backtrack(int row, char[][] board) {
    if (row == board.length) {
        result.add(buildBoard(board));
        return;
    }

    for (int col = 0; col < board.length; col++) {
        if (!isValid(row, col, board)) {
            continue;
        }

        board[row][col] = 'Q';
        backtrack(row + 1, board);
        board[row][col] = '.';
    }
}
```

记忆点：

- 通常一层处理一行。
- 放置前检查是否合法。
- 递归后恢复棋盘状态。

## 12. 同层去重总结

常见去重条件：

```java
if (i > startIndex && nums[i] == nums[i - 1]) {
    continue;
}
```

适合：子集 II、组合总和 II。

```java
if (i > 0 && nums[i] == nums[i - 1] && !used[i - 1]) {
    continue;
}
```

适合：全排列 II。

不能排序的题，例如递增子序列，用每层 `Set`：

```java
Set<Integer> usedInLevel = new HashSet<>();
for (int i = startIndex; i < nums.length; i++) {
    if (usedInLevel.contains(nums[i])) {
        continue;
    }
    usedInLevel.add(nums[i]);
}
```

## 13. 高频易错点

- 收集答案必须拷贝：`new ArrayList<>(path)`。
- 子集是进入递归就收集，组合是满足条件才收集。
- 组合和子集用 `startIndex`，排列用 `used[]`。
- 可重复选择递归传 `i`，不可重复选择递归传 `i + 1`。
- 去重前通常先排序，但子序列类题目不能排序。
- 修改了什么状态，回溯时就恢复什么状态。
- 剪枝只能剪掉确定不可能产生答案的分支，不能影响正确性。

## 14. 面试复述模板

可以按这个顺序讲：

1. 先说明递归函数含义。
2. 再说明 `path` 和 `result` 分别存什么。
3. 说明终止条件，以及什么时候收集答案。
4. 说明当前层如何枚举选择。
5. 说明递归参数如何变化。
6. 说明如何撤销选择。
7. 如果有重复元素，说明同层去重逻辑。

一句话总结：

```text
回溯就是在当前层枚举选择，选择后进入下一层，返回时撤销选择；不同题型的区别主要在收集答案的位置、下一层起点、以及是否需要去重和剪枝。
```
