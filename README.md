## 🧠 我的刷题方法

每道题建议按照下面的顺序完成：

1. 先独立思考题目。
2. 明确输入、输出和边界条件。
3. 尝试写出暴力解法。
4. 分析暴力解法可以优化的位置。
5. 判断题目属于哪一种算法类型。
6. 写出核心变量、状态或指针的含义。
7. 完成代码并分析复杂度。
8. 隔几天重新默写和复盘。

## 🧩 动态规划五部曲

解决动态规划问题时，可以按照下面五个步骤分析：

1. 确定 `dp` 数组以及下标的含义。
2. 确定递推公式。
3. 确定 `dp` 数组如何初始化。
4. 确定遍历顺序。
5. 举例推导 `dp` 数组。
